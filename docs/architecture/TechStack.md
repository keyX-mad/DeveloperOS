# DeveloperOS 技术栈（Tech Stack）

## 一、技术选型原则

- 优先选择成熟稳定的技术。
- 优先选择能够快速落地 MVP 的技术。
- 优先选择适合 AI 应用场景并具备长期扩展能力的技术。

------

# 二、前端技术栈

| 技术         | 用途          |
| ------------ | ------------- |
| Vue3         | 前端框架      |
| TypeScript   | 类型安全      |
| Pinia        | 状态管理      |
| Vue Router   | 路由管理      |
| Element Plus | UI 组件库     |
| Axios        | HTTP 请求     |
| Markdown-It  | Markdown 渲染 |

------

# 三、后端技术栈

| 技术            | 用途       |
| --------------- | ---------- |
| Spring Boot 3   | 应用框架   |
| Spring MVC      | Web 开发   |
| Spring AI       | 大模型集成 |
| MyBatis Plus    | ORM 框架   |
| Spring Security | 认证授权   |
| JWT             | 用户认证   |
