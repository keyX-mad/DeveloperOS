# DeveloperOS Core Values（核心价值观）

## 1. 用户成长第一（Growth First）

用户的成长是 DeveloperOS 存在的核心价值。所有功能设计和产品决策，都应优先考虑是否能够帮助用户持续成长。

------

## 2. AI 共同成长（Grow Together with AI）

AI 不仅是工具，更是陪伴开发者成长的长期伙伴。随着用户学习、实践和积累，AI 也能够持续理解用户的目标、知识体系和成长路径，与用户共同成长。

------

## 3. 成为理解用户的知音（Understand and Accompany）

DeveloperOS 希望成为真正理解开发者的 AI 伙伴。它能够记住用户的学习经历、项目经验和成长目标，在合适的时间提供建议、鼓励和帮助，让开发者在长期成长过程中不再孤单。
